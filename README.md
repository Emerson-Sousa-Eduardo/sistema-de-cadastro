# Sistema de Cadastro em Python

Projeto simples desenvolvido em Python para praticar estruturas de repetição, condicionais e validação de dados.

## Funcionalidades

* Cadastro de e-mail
* Criação e confirmação de senha
* Validação básica de e-mail
* Verificação de senha
* Mensagem de boas-vindas personalizada

## Tecnologias

* Python 3

## Como executar

1. Baixe ou clone o repositório.
2. Abra o terminal na pasta do projeto.
3. Execute o arquivo:

```bash
python cadastro.py
```

## Objetivo

Projeto criado para praticar conceitos fundamentais de programação em Python, como loops, validações de entrada e manipulação de variáveis.
