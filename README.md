# Sistema de Cadastro de Usuários em Python

##  Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais da linguagem Python, incluindo lógica de programação, estruturas de repetição, condicionais, tratamento de erros e manipulação de arquivos.

O sistema permite que usuários realizem cadastro e login, armazenando os dados em um arquivo de texto para posterior autenticação.

##  Funcionalidades

* Cadastro de novos usuários
* Validação de e-mail
* Validação de senha
* Confirmação de senha durante o cadastro
* Armazenamento de dados em arquivo `.txt`
* Sistema de login com verificação de credenciais
* Menu interativo no terminal
* Tratamento de opções inválidas
* Código comentado para facilitar manutenção e aprendizado

##  Tecnologias Utilizadas

* Python 3

##  Conceitos Praticados

* Variáveis
* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`while`, `for`)
* Manipulação de arquivos
* Tratamento de exceções (`try` e `except`)
* Validação de dados
* Organização e documentação de código
